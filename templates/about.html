About Us

CloudWiki Pro is a lightweight knowledge management platform built with Flask, MySQL, and deployed on AWS infrastructure.

The Project

This wiki was built as a hands-on project to demonstrate cloud architecture skills, including EC2, RDS, IAM, and security best practices. Users can register, log in, and contribute articles to a shared knowledge base.

Tech Stack

  • Backend: Python + Flask
  • Database: MySQL on Amazon RDS
  • Compute: Amazon EC2 (Ubuntu)
  • Frontend: Bootstrap + Jinja2 templates
